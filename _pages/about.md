## Modeling and Control Systems

Any systems including physical, biological, mechanical, etc. systems can be modeled using **dynamical mathematical model**. However, almost none of the system in the world can be modeled correctly. Therefore, scientists are looking for the best approximation which closely represents the physical, biological, mechanical, etc. systems. To do that, I am interested in **sparse (parsimonious) system identification** which describes the system by using simplest model. This simplification also eases the control design procedure. I have implementations of this concept in behavioral treatment design problems. Additionally, this model simplification might require **robust control**, especially for behavioral treatment design problems that I have experienced, for a specific system. To do that, I am also interested in developing robust model predictive control algorithms.
